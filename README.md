# MapStruct 

## O que é o MapStruct?
	MapStruct é um processador de anotações que é conectado ao compilador java, um gerador de código que simplifica o mapeamento dos objetos,
	entre diferentes modelos de objetos.
	Por exemplo: entidades para DTOs( Data Transfer Object) e vice-versa.

## Qual as vantagens de usar o MapStruct?
 - Em contraste com outras estruturas de mapeamento o MapSctruct gera mapeamentos de objetos em tempo de compilação,
 o que garante um alto desempenho e performance.
 - Para o desenvolvedor, escrever aquele código de mapeamento onde é uma tarefa grandiosa e propensa de erros,
 o MapStruct simplifica este trabalho, automatizando-o. 
 - O mapeamento gerado usa invocações de métodos simples, portanto, é rápido e fácil de entender
